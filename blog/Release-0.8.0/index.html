Release 0.8.0

Release notes for 10/26/15 V0.8.0.

Features

System Loading
Import a state from a previous run to pickup where you left off, decrease timesteps, or change the system forces all together.

Changes

Recovery File Syntax
The full system snapshots, recovery.txt, has been changed from CSV to space delimited.
System Configuration Snapshot
The initial snapshot of the system configuration sysConfig.txt has been renamed to systConfig.cfg for consistancy. Slightly more information is added to this file for reloading purposes.
Random number generation
The random number and uniform distibution generation have been changed from M19937 to a quicker custom generators. This should also C++11 dependencies from the engine core.

Bug Fixes

Snapshot Frequency
System snapshots will no longer skip the occasional frame.
ParticleBoundsError
Now shows the correct precision in particle position.
Output Folders
Fixed a failure to create output when prompted to input an output folder path manually.
