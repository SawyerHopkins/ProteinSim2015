Release 0.8.1

Release notes for 01/13/16 V0.8.1.

Features

Build Chain Automation
Added scripts to automate build maintenance.

Changes

Restructured Build Chain
The project now uses the NVIDIA Eclipse build structure.
Extract System Analysis Interface
Analysis functions, both runtime and post, are implemented using the new IAnalysisManager interface.

Bug Fixes

Single Particle Simulations
Fixed a division by zero error when simulating a single particle.
