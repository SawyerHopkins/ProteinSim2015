Open Source

ProteinSim is provided as a free open source project under MIT license. Clone or fork the latest version from our GitHub repository.

Programming

Built using only the latest C++ standard library for the largest system compatibility. No extra packages or libraries needed.

Parallelization

The ProteimSim library makes creating parallel dynamics simulations quick and easy. Support for CUDA device offloading is even available as an experimental release.
